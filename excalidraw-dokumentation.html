Excalidraw Integration - Dokumentation

Übersicht

Diese Dokumentation erklärt, wie du Excalidraw-Zeichnungen in deine GitHub Pages-Website einbinden kannst. Die Integration ermöglicht es, interaktive Zeichnungen anzuzeigen, zu bearbeiten (mit Passwortschutz) und als Bild zu speichern.

Hinweis: Die Integration verwendet die offizielle Excalidraw-Webanwendung über iframes, was eine einfache und zuverlässige Einbindung ermöglicht.

Excalidraw in eine bestehende Seite einbinden

Um eine Excalidraw-Zeichnung in eine bestehende HTML-Seite einzubinden, folge diesen Schritten:

1. Integration-Skript einbinden

Füge das Excalidraw-Integration-Skript in den Head-Bereich deiner HTML-Seite ein:

<script src="/assets/js/excalidraw-integration.js"></script>

2. Container-Element hinzufügen

Füge ein Container-Element mit einer eindeutigen ID in deine HTML-Seite ein:

<div id="meine-zeichnung"></div>

3. Excalidraw initialisieren

Füge ein Script-Tag hinzu, das die initExcalidraw-Funktion aufruft:

<script>
  document.addEventListener('DOMContentLoaded', function() {
    // Pfad zur Excalidraw-Datei (relativ zur HTML-Datei)
    const excalidrawFilePath = '/pfad/zu/deiner/zeichnung.excalidraw';
    
    // Excalidraw initialisieren
    initExcalidraw('meine-zeichnung', excalidrawFilePath, true);
  });
</script>

Die initExcalidraw-Funktion akzeptiert folgende Parameter:

Parameter Typ Beschreibung
containerId String Die ID des Container-Elements
excalidrawFilePath String Der Pfad zur .excalidraw-Datei
enablePasswordProtection Boolean Ob der Passwortschutz aktiviert werden soll (Standard: true)

Einen Excalidraw-Arbeitsbereich für Schüler erstellen

Um einen leeren Excalidraw-Arbeitsbereich für Schüler zu erstellen, folge diesen Schritten:

1. Container-Element hinzufügen

<div id="schueler-arbeitsbereich"></div>

2. Arbeitsbereich initialisieren

<script>
  document.addEventListener('DOMContentLoaded', function() {
    // Leeren Arbeitsbereich für Schüler erstellen
    createStudentWorkspace('schueler-arbeitsbereich');
  });
</script>

Neue Excalidraw-Dateien hinzufügen

Um neue Excalidraw-Dateien zu deiner Website hinzuzufügen:

1. Excalidraw-Datei erstellen

Erstelle eine Zeichnung mit Excalidraw und exportiere sie als .excalidraw-Datei.

2. Datei in das Repository hochladen

Lade die .excalidraw-Datei in dein GitHub-Repository hoch, z.B. in einen Ordner wie /assets/excalidraw/.

3. Datei in einer Seite verwenden

Verwende die initExcalidraw-Funktion, um die Datei in einer Seite anzuzeigen (siehe oben).

Wichtig: Stelle sicher, dass der Pfad zur .excalidraw-Datei korrekt ist. Der Pfad ist relativ zum Stammverzeichnis der Website.

Passwortschutz anpassen

Das Standard-Passwort für den Bearbeitungsmodus ist lehrer2024. Um das Passwort zu ändern, musst du die excalidraw-integration.js-Datei bearbeiten:

// In assets/js/excalidraw-integration.js
const PASSWORD = "dein-neues-passwort";

Sicherheitshinweis: Beachte, dass ein clientseitiger Passwortschutz nur einen grundlegenden Schutz bietet. Das Passwort ist im JavaScript-Code sichtbar und kann von technisch versierten Benutzern umgangen werden.

Teilen von Excalidraw-Zeichnungen

Es gibt mehrere Möglichkeiten, wie Schüler ihre Excalidraw-Zeichnungen mit dir teilen können:

1. Lokales Speichern und Teilen

Schüler können ihre Zeichnungen lokal speichern und dann über verschiedene Wege teilen:

  • E-Mail: Die .excalidraw-Datei als Anhang per E-Mail senden
  • Cloud-Dienste: Die Datei in Google Drive, OneDrive oder ähnlichen Diensten hochladen und den Link teilen
  • Lernplattformen: Die Datei in Moodle, MS Teams oder anderen Lernplattformen hochladen

2. Kollaborative Bearbeitung

Für kollaboratives Arbeiten können folgende Optionen genutzt werden:

  • Stelle sicher, dass du das richtige Passwort eingibst.
  • Überprüfe, ob die excalidraw-integration.js-Datei korrekt geladen wird.
