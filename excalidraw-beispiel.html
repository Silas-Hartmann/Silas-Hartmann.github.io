Excalidraw Beispiel - Überschwemmung Ahrtal

Diese Seite demonstriert die Integration von Excalidraw in GitHub Pages. Die Zeichnung zeigt ein Tafelbild zum Thema "Überschwemmung".

Um die Zeichnung zu bearbeiten, geben Sie das Passwort ein und klicken Sie auf "Bearbeiten".

Hinweis: Das Passwort für dieses Beispiel ist "lehrer2024".

Wie funktioniert es?

Die Excalidraw-Integration ermöglicht es, interaktive Zeichnungen in GitHub Pages einzubetten. Die Zeichnungen können:

  • Im schreibgeschützten Modus angezeigt werden (Standard)
  • Nach Passworteingabe bearbeitet werden
  • Als Bild gespeichert werden
